Git

  • git status: checks what branch your currently working on
  • git checkout -b branch-name : creates a new branch and switches to it
  • git add [file] : add a file as it looks to the next commit stage
  • git reset [file] : unstage a file while retaining changes in working directory
  • git diff : dii of what is changed but not staged
  • git diff --staged : diff of what is staged but not yet commited
  • git commit -m [descriptive message] : commit new staged content as a new commit snapshot
  • git branch : list your branches
  • git branch : [branch-name] create new branch at the current commit
  • git checkout : Switch to another branch and check it out into your working directory
  • git merge [branch] : Merge the specified branches history into the current one
  • git log : Show all commits in the current branche's history
  • git init : Initialize an existing directory as a GIT repo
  • git clone [url] : Retrieve an entire repo from a hosted location via URL
  • git log branchB. .branchA : show commits on branchA not on branchB
  • git log --follow [file] : show the commits that changed file even across renames
  • git diff branchB. . .branchA : show the diff of what is in the branches
  • git show [SHA] :Show any object in GIT in human readable format
